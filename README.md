🤖 Gemini AI Chatbot

An AI-powered chatbot built using **Python** and **Google Gemini API** that can understand natural language queries and respond intelligently in real time. This project is suitable for college submissions, resumes, and GitHub portfolios.

---------------------------------------------------------------------------------------------------------------------------------------------------------

 🚀 Features

* Conversational AI using Google Gemini
* Context-aware responses (chat memory)
* Fast and lightweight model (`gemini-1.5-flash`)
* Error handling for safe API usage
* Simple terminal-based interface (easy to extend to web/GUI)

---------------------------------------------------------------------------------------------------------------------------------------------------------

🛠️ Tech Stack

* **Language:** Python
* **AI Model:** Google Gemini API
* **Libraries:** google-generativeai
* **Tools:** Git, GitHub

⚙️ Installation & Setup

 1️⃣ Clone the repository

```
bash
git clone https://github.com/your-username/Gemini-AI-Chatbot.git
cd Gemini-AI-Chatbot
```
---------------------------------------------------------------------------------------------------------------------------------------------------------
2️⃣ Install dependencies

```
bash
pip install google-generativeai
```
---------------------------------------------------------------------------------------------------------------------------------------------------------
3️⃣ Get Gemini API Key

* Visit **Google AI Studio**
* Generate an API key

⚠️ **Do not expose your API key publicly**

Set API key in code or environment variable:

```
python
import os
os.environ["GEMINI_API_KEY"] = "YOUR_API_KEY"

```

---------------------------------------------------------------------------------------------------------------------------------------------------------

▶️ Run the Chatbot

_bash_
python chatbot.py


Type your message and chat with the AI. Type `exit` to quit.
